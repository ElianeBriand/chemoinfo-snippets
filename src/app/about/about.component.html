Chemoinformatics Snippets
About

A small website to gather useful snippets of chemoinformatics code.

Licensing of the snippets

To the extent possible under law, the person who associated CC0 with each snippet has waived all copyright and related or neighboring rights to the snippet.

Contributing

Source code (including the snippets) on GitHub.

Contribution welcome both in content and js/html code. You can submit a new snippet either by opening a snippet issue or making a pull request. As the snippet lists are stored in standard YAML, it is relatively easy to contribute.

You can use the following tool to preview the result for a given YAML-formatted snippet: YAML previewer
